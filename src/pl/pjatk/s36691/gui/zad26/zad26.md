# Zadanie 26 - Enum Color i mieszanie kolorów

Należy stworzyć enum `Color` z podstawowymi kolorami.

## Wymagania
- Dodać metodę statyczną `mixColors(Color a, Color b)`.
- Metoda ma zwracać kolor powstały po zmieszaniu dwóch różnych kolorów podstawowych.

## Główne zagadnienia
- enum
- logika warunkowa
- metody statyczne
