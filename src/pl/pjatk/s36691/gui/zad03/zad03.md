# Zadanie 03 - Pacjenci i polimorfizm

Należy zdefiniować klasy potrzebne do działania przykładowego programu z pacjentami.

## Wymagania
- Przygotować typ `Pacjent`.
- Stworzyć klasy konkretnych pacjentów, np. `ChoryNaGlowe`, `ChoryNaNoge`, `ChoryNaDyspepsje`.
- Zapewnić polimorficzne działanie metod:
  - `choroba()`
  - `leczenie()`
- Program ma wypisywać dane pacjenta, chorobę i zastosowane leczenie.

## Główne zagadnienia
- dziedziczenie
- polimorfizm
- klasy abstrakcyjne lub interfejsy
