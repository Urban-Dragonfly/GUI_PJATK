# Zadanie 06 - Komputery i laptopy

Należy uzupełnić brakujące klasy, pola, konstruktory i elementy dziedziczenia w hierarchii komputerów.

## Wymagania
- Stworzyć klasy `Computer`, `PersonalComputer`, `Laptop`.
- Wszystkie pola mają być prywatne.
- Program ma wypisywać obiekty zgodnie z podanym formatem.
- Dla laptopa należy przygotować metodę sprawdzającą wagę i rzucającą wyjątek przy zbyt dużej wartości.

## Główne zagadnienia
- dziedziczenie
- prywatne pola
- `toString()`
- wyjątki
