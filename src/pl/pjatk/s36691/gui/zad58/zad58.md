# Zadanie 58 - Gra memory 4x4

Należy stworzyć prostą grę memory w układzie 4x4.

## Wymagania
- Przygotować 8 obrazków i użyć ich parami.
- Zaimportować obrazki do aplikacji.
- Dodać przycisk resetu gry.

## Główne zagadnienia
- logika gry
- GUI
- obsługa kliknięć

## Komentarze

Projekt znacząco wykracza poza zakres tego zadania.

Celem rozszerzenia zadania było stworzenie gry, którą można pochwalić się ukochanym osobom, jako podsumowanie pierwszego roku na studiach.

Z uwagi na to na aktualnym etapie projekt wymaga refaktoryzacji kodu, uporządkowania architektury MVC i rozdzielenia zależności.
