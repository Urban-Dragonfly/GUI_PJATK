# Zadanie 33 - Lista zwierząt i filtrowanie po wysokości

Należy stworzyć klasę `Animal` z podstawowymi polami opisującymi zwierzę.

## Wymagania
- Dodać pola:
  - `name`
  - `weight`
  - `height`
  - `topSpeed`
- Wypełnić listę obiektami.
- Stworzyć drugą listę nazw zwierząt wyższych niż 1 metr:
  - raz przy pomocy `for-each`,
  - raz przy pomocy strumieni.

## Główne zagadnienia
- listy
- stream API
- `map()`
- `filter()`
