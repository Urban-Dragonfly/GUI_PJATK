# Zadanie 65 - Prosty edytor tekstu w JavaFX

Należy przygotować prosty edytor tekstu w JavaFX.

## Wymagania
- Umieścić komponent pola tekstowego.
- Ustalić cechy wizualne:
  - kolor tła,
  - kolor pisma,
  - rodzaj,
  - styl,
  - rozmiar.
- Zastosować style CSS.

## Główne zagadnienia
- JavaFX
- stylowanie CSS
- pola tekstowe
