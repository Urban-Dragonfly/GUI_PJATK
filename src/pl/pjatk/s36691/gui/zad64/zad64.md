# Zadanie 64 - JavaFX Hello world z CSS

Należy napisać program w JavaFX z przyciskiem zmieniającym napis.

## Wymagania
- Początkowo przycisk pokazuje `Hello world!`.
- Po kliknięciu napis zmienia się na `Hi`.
- Kolejne kliknięcie przywraca pierwszy napis.
- Zastosować style CSS.

## Główne zagadnienia
- JavaFX
- obsługa zdarzeń
- CSS
