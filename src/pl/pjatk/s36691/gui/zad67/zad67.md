# Zadanie 67 - Wyświetlanie strony internetowej w JavaFX

Należy napisać program w JavaFX wyświetlający stronę internetową.

## Wymagania
- Jako przykład można użyć `pja.edu.pl`.
- Strona ma być przewijalna.
- Ma wypełniać całe okno aplikacji.

## Główne zagadnienia
- JavaFX
- komponent webowy
- osadzanie treści WWW
